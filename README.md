# Social Multiplication

## Prerequisites
* SpringBoot 1.5.21
* Java 8 (1.8)

## Installation
### Java JDK (not JRE)
1.  Download JDK (https://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html) and run installer

## Run
* ```./mvnw spring-boot:run```

## Tests
* ```./mvnw test```
* ```./mvnw -Dtest=MultiplicationServiceTest test```
